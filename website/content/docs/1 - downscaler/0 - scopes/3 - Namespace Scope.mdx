# Namespace Scope

The Namespace Scope holds configurations set at the namespace level.
It overrides values already defined inside the
default scope, the Env Scope, the CLI Scope, but values defined inside the namespace scope can be overidden by values
defined inside the Workload Scope.
The namespace Scope only holds values.
