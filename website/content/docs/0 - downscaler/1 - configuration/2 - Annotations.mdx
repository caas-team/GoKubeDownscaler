# Annotations

[Values][docs-values] can be set via annotations on the [namespace][docs-namespace-layer] or [workload layer][docs-workload-layer].

## Valid Values

- [downscaler/downscale-period][docs-values-downscale-period]
- [downscaler/downtime][docs-values-downtime]
- [downscaler/upscale-period][docs-values-upscale-period]
- [downscaler/uptime][docs-values-uptime]
- [downscaler/exclude][docs-values-exclude]
- [downscaler/exclude-until][docs-values-exclude-until]
- [downscaler/force-uptime][docs-values-force-uptime]
- [downscaler/force-downtime][docs-values-force-downtime]
- [downscaler/downscale-replicas][docs-values-downscale-replicas]
- [downscaler/grace-period][docs-values-grace-period]

## Usage

```yaml
apiVersion: v1
kind: Namespace
metadata:
  name: example-namespace
  annotations:
    downscaler/downscale-replicas: "1"
    downscaler/exclude: "false"
---
apiVersion: apps/v1
kind: Deployment
metadata:
  name: example-deployment
  annotations:
    downscaler/uptime: "Mon-Fri 08:00-20:00 UTC"
spec:
  replicas: 3
  selector:
    matchLabels:
      app: example-app
  template:
    metadata:
      labels:
        app: example-app
    spec:
      containers:
        - name: example-app-container
          image: nginx:latest
```
