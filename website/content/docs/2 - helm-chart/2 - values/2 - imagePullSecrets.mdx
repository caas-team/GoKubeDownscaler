# imagePullSecrets

The `imagePullSecrets` value is a list of [Kubernetes secrets](https://kubernetes.io/docs/concepts/configuration/secret/)
that are needed to [pull an image from a private registry](https://kubernetes.io/docs/tasks/configure-pod-container/pull-image-private-registry/).

:::info

The default value for `imagePullSecrets` is empty:

```yaml
imagePullSecrets: []
```

:::
