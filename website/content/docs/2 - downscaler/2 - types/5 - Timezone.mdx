# Timezone

Timezone is a value that can be set to specify the global timezone in which the GoKubeDownscaler should operate when
evaluating timespans without a specific timezone set.

## Syntax

Timezone is a value from the [IANA Time Zone database](https://www.iana.org/time-zones).

```text
America/Phoenix # set the global timezone to America/Phoenix timezone
```
