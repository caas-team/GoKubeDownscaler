# configMap

The name of the configmap for the go-kube-downscaler.

extraConfig adds additional specified environment variables to the ConfigMap.

> ℹ️ **The default values for `configMap` are:**
>
> ```yaml
> configMap:
>   name: go-kube-downscaler
>   extraConfig: ""
> ```
