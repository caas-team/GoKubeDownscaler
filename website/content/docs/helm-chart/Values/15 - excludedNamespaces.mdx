# excludedNamespaces

A list of namespaces that are supposed to be excluded from the downscaling process of the go-kube-downscaler.

:::info
The default value for `excludedNamespaces` is:

```yaml
excludedNamespaces:
  - kube-downscaler
  - kube-system
```

:::
